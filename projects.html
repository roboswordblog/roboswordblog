CSV File Editor in Java

This is a csv file editor made in java. You can drop NaN values in the row, and you can replace values. You can also save the editted file to your memory. I mostly made this application so that I could use it for editting data and seeing the result in real time. Its in my github.

Multiplayer Agar.IO

This is a multiplayer agar.io game made in python using pygame and sockets. It can only run on local host, and it was made for me and my friends to play at school. It only works if the server is on the same wifi as all the clients. You can also host and play your own games, and you can host mulitple games by hosting it on different ports.

Text Sentiment Classifier

This is a text sentiment classfier made using pytorch. I encoded the data using tf-idf. It predicts whether the text is negative, positive, or neutral. It has a pretty high accuracy, even though it is not industrial standard.
Cricket Runs Predictor
This was one of my first machine learning projects with pytorch. I made a nn to predict cricket runs of a player based on the player's stats. When I refer to runs, I am meaning runs per match. Free Typing
A typing practice application built with Python and Pygame that tracks user performance in real time. The app measures words per minute (WPM) and accuracy, provides immediate visual feedback on keystrokes, and includes a replay feature that lets users review which words were typed correctly or incorrectly. This project helped me explore real-time input handling, game loops, and performance tracking, while also highlighting the challenges of building interactive applications with Pygame.
